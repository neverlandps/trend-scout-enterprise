ARCHITECTURE

成熟可靠的技术栈

每一层都选择经过生产验证的主流技术，自托管部署，数据完全掌握在自己手中。

FastAPI SQLAlchemy PostgreSQL Celery Redis LangGraph React 18 Fluent UI Docker SPFx
架构说明：React + Fluent UI 前端通过 REST API 与 FastAPI 后端通信；Celery + Redis 承担定时扫描与报告渲染等异步任务；LangGraph 编排 TrendAnalystAgent 的多步推理流程；PostgreSQL 存储业务数据与向量索引，SPFx Web Part 经由只读嵌入令牌将仪表盘安全嵌入 SharePoint。
GETTING STARTED

三步完成部署

一条 docker compose 命令拉起完整服务栈（API、Worker、调度器、Redis），五分钟内看到第一个趋势仪表盘。

1

启动服务栈

# 拉取预构建镜像并启动
cd deployment
GITHUB_REPOSITORY_OWNER=your-org \
  docker compose up -d
2

配置安全密钥

# 生产环境必需，缺省将拒绝启动
export SECRET_KEY=$(openssl rand -hex 32)
3

打开工作台

# 浏览器访问，开始使用
http://localhost:5173

# API 文档
http://localhost:8000/docs
在 GitHub 上查看源码
